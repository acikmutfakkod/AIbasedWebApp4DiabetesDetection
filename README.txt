Kodu Çalıştırma Adımları:

1) pip ile gerekli kütüphaneleri aşağıdaki gibi yükleyelim:
	pip install pandas
	pip install scikit-learn
	pip install Pillow
	pip install streamlit
2) "https://www.kaggle.com/saurabh00007/diabetescsv/data" adresinden diabetes.csv dosyasını indirelim ve DiabetesDataset.csv adıyla proje dizinimizin içerisine taşıyalım.
3) Kodun bilgisayarımızda çalışması için main.py dosyasında iki yerde düzenleme yapmamız gerekiyor. Bunlar resmi yüklediğimiz 19.satır ve de veri setini okuduğumuz 23.satır. Her ikisinde de dosya yoluna proje klasörümüzün olduğu yeri yazıp, main.py dosyasını kaydedelim.
4) Kodu çalıştırmak ve Yapay Zeka tabanlı Web Uygulamamıza erişmek için tekrar komut penceremize dönelim ve aşağıdaki komutu çalıştıralım.
	streamlit run "main.py dosyasının dosya yolu"
5) Gelen e-posta girişini boş bırakarak geçebiliriz.
6) Uygulamanın internet tarayıcımızda açılmış olması lazım.
